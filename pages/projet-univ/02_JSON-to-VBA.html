🗄️ SAÉ - Écriture et lecture de fichiers de données

Création d'un outil de converstion de fichier JSON en CSV en python (rendu en .exe).

Universitaire S1 - 2025/2026 Terminé
Présentation

Le 1er semmestre a était une remise a niveaux en informatique et programation. Nous avons donc revue (ou vue pour certain) les base de l'algorithime ansi que la lecture et l'écriture de données. Pour nous entré sur ces nouvelle compétence, nous avons réaliser ce projet.

Avec python, nous devion parcourire un fichier JSON, extrarire uniquement sertaine information, puis les écrire dans un fichier CSV. Il y avais donc l'utilisation de lsite et dictionaire python. Le fichier JSON était un fichier résultat de capeteur de CO2 de la région. Nous devion donc comprendre la structure du JSON avant tout manipulation.

Pour la part, étant déjà initier a la programation, je me suis amusé a allez un peux plus loin en ajoutant une interface graphique (minimaliste) pour selectioner le fichier d'entré (JSON) et le nomage du fichier résutlat (CSV) avec gestion des chemin absolue ou relatif. Le livrable a était rendu avec un .exe. Le code a était fait en Programation Orienté Objet (POO).

Technologies
Python
POO. Parcour de fichier (lecture et écriture). Interface graphique (kivy). Conversion du script en .exe.
JSON / CSS
Type de structuration de données. Lecture et ecriture
Détails
Fonctionnalités principales
  • Choix du fichier d'entré et sorti
  • Lecture, extraction du fichier JSON
  • Ecriture des données demandé dans le fichier CSV
  • Application forme exécutable (.exe)
  • Lever d'exception pour plus de robustése
Difficultés rencontrées

Je ne conaissé que peux la bibliotéque kivy (interface graphique multiplatforme - pyhton). Cela ma données l'occastion de travaillé cette compétence. J'ai également pu me former sur le passage d'un scripte python (.py) a un exécutable (.exe)
