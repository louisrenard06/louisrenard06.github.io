🗄️ Système de Gestion
de Base de Données

SGBD simplifié en Java — transactions, indexation B-tree et interface SQL intégrée.

Universitaire 2024 Terminé
Présentation

Dans le cadre de ce projet universitaire, l'objectif était de concevoir et implémenter un Système de Gestion de Base de Données (SGBD) simplifié en Java, sans recourir à une bibliothèque externe. Ce projet m'a permis de comprendre en profondeur le fonctionnement interne d'un moteur de base de données.

Le système supporte un sous-ensemble du langage SQL : les requêtes SELECT, INSERT, UPDATE et DELETE, ainsi que la création et suppression de tables. La gestion des transactions garantit l'atomicité et la cohérence des opérations, avec support du rollback.

Détails
Fonctionnalités principales
  • Parsing et exécution de requêtes SQL (SELECT, INSERT, UPDATE, DELETE)
  • Gestion des transactions avec support rollback et commit
  • Indexation B-tree pour optimisation des recherches
  • Persistance des données sur disque en format binaire
  • Interface ligne de commande interactive
Architecture

L'architecture suit un pattern MVC adapté aux bases de données : le Parser extrait l'AST de la requête SQL, le Planner génère le plan d'exécution, et l'Executor interagit avec le StorageManager pour les opérations disque.

Difficultés rencontrées

La gestion de la concurrence et l'implémentation correcte du protocole de verrouillage en deux phases (2PL) ont été les aspects les plus complexes. La sérialisation des données binaires a également nécessité une attention particulière pour la portabilité.
